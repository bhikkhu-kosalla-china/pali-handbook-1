# 非主格受动者 → 动词(IOV)

* English： **I**nactive **O**bject **V**erb Relation
* 中文： 受动者-动词关系
* 巴利语： avuttakamma - kiriyā
 

一个宾格宾语与一个动词的主动态或绝对态相关联。
