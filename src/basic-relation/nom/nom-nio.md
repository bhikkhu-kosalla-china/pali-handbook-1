# 主语-表语关系

|语种|名称|
|-|-|
|巴利语|tulyattha-liṅgattha|
|中文|主语-表语|
|English|Nominal Identity (Ordinary) Relation|


#### 例句：so ācariyo

<div class="mermaid">
graph LR
A("so<br>他<br>阳·单·主")
B("ācariyo<br>老师<br>阳·单·主")
A--"NIO<br>主 ➡ 表"-->B
</div>

- 译文：他（是）老师。
> 该例句中：**性**、**数**、**格**相同

#### 例句：cittāni vīsati
<div class="mermaid">
