# 参照→比较级（COC）

COC=**Co**ntrastive **C**omparison Relation
Vibhattāpādāna - Vibhattāpādānin

作为形容词比较级的参照，做来源格变化。

例句：so asm**āhi** pāpiyataro<br>
他**比**我们更差。
