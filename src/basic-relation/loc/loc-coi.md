# 集合-个体元素关系

* 巴利： niddhāraṇasamudāya - nidhāraṇīya
* 中文： 集合-个体元素关系

分类：名词 → 名词

这种关系类型可能出现在**属格**和**处格**两种情况。

rājā manussesu
