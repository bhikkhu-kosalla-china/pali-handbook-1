# 集合-个体元素关系

* 巴利： niddhāraṇasamudāya - nidhāraṇīya
* 中文： 集合-个体元素关系
* English： Collection -Individual Relation (COI)

这里的这种关系类型在**属格**的时候也是有可能的。

This type of relation is also possible here as with Genitive case.

rājā manussesu
- （那位/一位） **在**男人们**中**的国王...
- (The/A) king, **among/of** men...
