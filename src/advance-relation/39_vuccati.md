# **39. Usage of** *vuccati* 
In some sentences with the verb *vuccati*, the Quotation Marker *iti* has to be added  because it has not been expressed yet. 

**Ex**. ayam balo (*iti*) vuccati. **Trs**. He is mentioned as,  a fool”, i.e. termed a fool.
